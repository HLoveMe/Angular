1: npm install 加载 node——module
	>修改package.json 获取最新的 库
	
2: src/app下 为组目录

3: src/assets 为资源文件
		图片/js/css
		
		css{
			background:"/assets/img/aaa.png"
		}
		
4:npm start 启动开发

5:打包

	# development
		npm run build:dev
	# production (jit)
		npm run build:prod
		npm run build:aot:prod 
	# AoT
		npm run build:aot
		
		
		
		
6 测试。。。其他功能
